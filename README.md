# 📖 Overview

AI Deadline Guardian is an intelligent productivity assistant that analyzes tasks, predicts deadline risks, generates optimized schedules, and helps users recover from missed timelines using AI-powered rescue planning.

Instead of simply tracking tasks, the platform actively coaches users to complete projects before deadlines.

# 🚀 Installation

Clone the repository

```bash
git clone https://github.com/amishbuilds/Ai-Deadline-Guardian.git
```

Navigate into the project

```bash
cd AI-Deadline-Guardian
```

Install dependencies

```bash
npm install
```

Create a `.env` file

```env
GEMINI_API_KEY=YOUR_API_KEY
```

Run development server

```bash
npm run dev
```

Build for production

```bash
npm run build
```

Start production server

```bash
npm start
```
