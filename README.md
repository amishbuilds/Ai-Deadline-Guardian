# рџљЂ Installation

Clone the repository

```bash
git clone https://github.com/amishbuilds/Ai-Deadline-Guardian.git
```

Navigate into the project

```bash
cd AI-Deadline-Guardian
```

Install dependencies

```bash
npm install
```

Create a `.env` file

```env
GEMINI_API_KEY=YOUR_API_KEY
```

Run development server

```bash
npm run dev
```

Build for production

```bash
npm run build
```

Start production server

```bash
npm start
```
